最后是一些参数：

  • -a 列出打开文件存在的进程
  • -c<进程名> 列出指定进程所打开的文件
  • -g 列出GID号进程详情
  • -d<文件号> 列出占用该文件号的进程
  • +d<目录> 列出目录下被打开的文件
  • +D<目录> 递归列出目录下被打开的文件
  • -i<条件> 列出符合条件的进程。（4、6、协议、:端口、 @ip ）
  • -p<进程号> 列出指定进程号所打开的文件
  • -u 列出UID号进程详情
